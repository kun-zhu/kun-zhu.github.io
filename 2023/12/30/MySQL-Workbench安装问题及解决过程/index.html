MySQL Workbench安装问题及解决过程

发表于 2023-12-30

在安装 MySQL Workbench 过程中遇到了一个奇怪的问题。以下是问题的详细描述以及最终的解决方案。

问题描述

  1. 从官网下载了 MySQL Workbench 的安装包（mysql-workbench-community-8.0.33-winx64）进行安装。
  2. 安装时报错，提示需要先安装 Visual C++ 2019。
  3. 从安装页面跳转到 MySQL 的官网 MySQL 官网 获取对应的下载链接。如图1。
  4. 从 MySQL 官网跳转到 Microsoft Office 的官网 Microsoft Office 官网 下载 Visual C++ 2019。如图2。
  5. 安装 Visual C++ 2019 后，仍然遇到相同的错误，多次尝试均未成功。

解决方法

最后采用了另一种方法，直接下载了 MySQL Server 的安装包（mysql-installer-community-8.0.35.0）。

  1. 下载地址：MySQL Installer
  2. 这个安装包不仅可以用来安装 MySQL Server，还可以安装其他组件，例如 MySQL Shell、MySQL Workbench 等。
  3. 在使用这个安装包进行安装时，同样提示缺少 Visual C++ 2019。
  4. 但这个安装包可以自动加载并安装 Visual C++。点击安装，成功安装了 Visual C++ 的包。
  5. 随后再安装 MySQL Workbench，成功完成安装。

区别分析

仔细对比两种安装方式，发现自己从 Microsoft 官网 下载的安装包是 Visual C++ 2015-2022，而使用 mysql-installer-community-8.0.35.0 自动安装的是 Visual C++ 2015-2019。
这似乎是一个潜在的 bug，因为上述 Microsoft 官网并不提供 Visual C++ 2015-2019 版本。
